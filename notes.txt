        Branches
        -=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-

             a branch is a parelell copy of our repo.

             a repo can lie on a single branch at a time.

             we can switch between branches freely.

             git branch

             git branch branchName      ---> add a new branch

             git checkout branchName    ---> will switch from current branch to the one specified

             git checkout -b newBranchName --> will add and switch to that branch.

             git merge branchName          ---> the current branch will be merged with the specified branch

             git branch -D branchName      ----> Removes a branch
